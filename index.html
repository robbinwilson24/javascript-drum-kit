Welcome to the this virtual Drum Kit, powered by JavaScipt!

This is a simple but fun functioning virtual drum kit.

Simply interact with the different components of the drum by clicking on them or use the corresponding keys displayed on your keyboard to play the drum sounds. From the dynamic crash cymbal to the resonant kick drum, embrace your inner drummer and give it a go.

Be advised that this virtual drum set is strictly intended for demonstration purposes and is not intended for commercial use. No financial gain is recieved from this project.


The sound effects utilised in this drum set are sourced from 99sounds.org.

The functionality of the virtual drums have been inspired and assisted by the work of Wes Bos, and his creation of virtual drums (at wesbos.com ), that provided some the starter code required for this finished product.
